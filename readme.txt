JLokalize     version 1.1

Free, open source, translations editor for Java programs.

release date  november 2011
website       jlokalize.sourceforge.net
author        Trilarion (trilarion@users.sourceforge.net)



1. Requirements

Java runtime (http://java.com/en/download/manual.jsp)


2. Starting the programm

Unzip the package to a folder of your choice.

Double (Windows) or single click (Ubuntu, ..) on JLokalize.jar should start the programm.

Troubleshooting:
See these tips (http://www.wikihow.com/Run-a-.Jar-Java-File) and make sure the execution flag is set under Linux.
